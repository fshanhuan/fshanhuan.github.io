表格

  • tr：tr 是 table row 的缩写，表示表格的一行。

  • td：td 是 table data 的缩写，表示表格的数据单元格。

  • th：th 是 table header的缩写，表示表格的表头单元格

标签属性

链接与资源类

这类属性通常出现在 <a>（超链接）或 <link> 标签中。

  • **href (Hypertext Reference)**：
    • 用法：<a href="https://google.com">
    • 作用：定义链接的目标地址。
  • **target**：
    • 用法：<a href="/" target="_blank">
    • 作用：决定在哪里打开链接。_blank 表示在新窗口打开，默认是在当前窗口打开。

图像与媒体类

这类属性出现在 <img>（图片）、<video> 或 <audio> 中。

  • **src (Source)**：
    • 用法：<img src="cat.jpg">
    • 作用：图片的“实体文件”存放路径。没有它，图片就显示不出来。
  • **alt (Alternative text)**：
    • 用法：<img src="cat.jpg" alt="一只可爱的橘猫">
    • 作用：极其重要！ 如果图片加载失败，网页会显示这段文字；同时它也能帮视障人士通过读屏软件理解图片内容。
  • **width / height**：
    • 用法：<img src="logo.png" width="200">
    • 作用：直接在 HTML 里规定图片的尺寸（单位通常是像素）。

表单与交互类

当你写输入框、按钮时，这些属性是核心。

  • **type**：
    • 用法：<input type="password">
    • 作用：决定输入框的性质。可以是 text（文本）、password（密码，显示点点）、checkbox（多选框）等。
  • **placeholder**：
    • 用法：<input type="text" placeholder="请输入你的账号">
    • 作用：在输入框没写字时显示的“提示灰字”。
  • **value**：
    • 用法：<input type="button" value="点我提交">
    • 作用：定义标签上显示的默认值或按钮上的文字。

通用标识类

这类属性几乎可以放在任何标签上，是后续学习 CSS 和 JavaScript 的基础。

  • **id**：

    • 用法：<div id="header">
    • 作用：全网页唯一的编号。就像身份证号，一个页面里同一个 id 只能用一次。

  • **class**：

    • 用法：<p class="blue-text">
    • 作用：分类标识。就像校服，很多个标签可以共用同一个 class，方便我们一次性给它们设置相同的样式。

  • **style**：

    • 用法：<h1 style="color: red;">

    • 作用：直接在标签上写样式（虽然不建议写太多，但在快速测试时很好用）。
